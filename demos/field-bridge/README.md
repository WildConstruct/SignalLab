# Field Bridge (demo) — Signal Rack × Etheros

Two engines, one look. The Signal Rack driver's `n` **routes onto an Etheros-style
recipe parameter** (`warp.amount` / `motion.flowStrength` / `shape.thresholdLow` /
`structure.baseScale`), rendered as a field heatmap via `shared/etheros-lite.js` (a
CPU mirror of the Etheros recipe pipeline). Demonstrates the field⟷signal
composition from `docs/web-demos/etheros-convergence.md`.

> etheros-lite is a web-demo mirror, not the real Dawn engine — it proves the
> composition shape, not pixel parity.

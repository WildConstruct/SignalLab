across the window with a glowing head dot on the live value. Best surface to feel
the Combine selector (X·Y·diff…). Roadmap (vectorscope, spiro, rose) in
`docs/web-demos/path-scope.md`.

`python3 -m http.server` → `/demos/path-scope/`

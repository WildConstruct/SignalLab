# Path & Scope (demo)
The raw signal **as geometry**. Current variants include wave-runner,
vectorscope, spirograph, rose curve, 2D/3D Lissajous, and strange attractors.
This is the clearest surface for feeling the Combine selector (`X·Y`, `diff`,
`min`, `max`) and the future Signal Rack guide-layer scope.

`python3 -m http.server` → `/demos/path-scope/`

- **Structure** — path type, line weight, turns/petals, radius, Lissajous ratios,
  attractor system, and 3D rotation.
- **Signal** — source/rate/**combine**/speed.
- **Shaping** — curve/head intensity and signal-perturbed phase or attractor
  parameters depending on the selected path.

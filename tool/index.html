Signal Rack — Signal Lab

Build a signal, then watch it drive things. Everything you see runs on the same WebGPU engine the After-Effects bridge uses — so a look you dial in here can be reproduced exactly, even copied out as a working AE expression.

1 · Make a signal

Generator has two channels, X and Y. Each is a source (Sine, Square, Saw, Triangle, Pulse, Noise, Random Walk) shaped by Rate, Phase, Amount, Offset, Smooth and a Seed (re-rolls the random sources and shifts the wave). The little “o” next to Amount/Phase oscillates that value slowly.

Channel Y can Sync to X (becomes X) or Invert (mirror X) (becomes 1−X) for clean symmetric figures.

2 · Shape it

  • Processor (shapes X): gain/bias, saturate, warp, fold, gate, quantize, lag, invert, rectify.
  • Window: a feathered region that fades the signal in/out across the scope.
  • Sidechain: let X modulate Y’s amplitude, rate (FM) or phase.
  • Distort: a gentle third signal that phase-bends both channels.

3 · See it

Tabs switch the scope: Waveform, Bars, Vectorscope, Polar, Heat, and Applied — where the signal drives something:

  • Transform / Visual / Meter: scale, rotate, opacity, glow, gauges, LED ladders…
  • FUI: HUD widgets — radar, oscilloscope, gyro rings, waveform ring, data stream, and the firing ones (Synapse net, Data packets, Processor die) that light up where the signal is active. Fire ≥ sets how active it must be; Seed reshapes the synapse map.
  • Transitions: a playhead ramps 0→100% across the signal (turn on Freeze to hold the curve) — loading bars, wipes, text reveals.
  • Particles / Text / Path: fountains, confetti, kinetic type, Lissajous, spirographs…

Drive picks how X and Y combine into the value (Channel X, Y, X×Y ring, mix, difference, min, max).

4 · Use it

  • ƒ AE expression (in Applied): the full, self-contained After-Effects expression — copy it onto any property; it reproduces the exact driver, no plugin needed.
  • Record video: capture the visualization to an mp4/webm.
  • Copy link / Export JSON: the whole setup travels in a URL or a recipe file.
  • 🎲 Surprise me: roll a tasteful random rig.

Display

Speed (÷2 / 1×) is a master time scale. Window = scope length, Persist = trail, Freeze = hold, CRT = Cathode tube look, Y opacity = dim channel Y.

Shortcuts

Space/F freeze · R surprise · P present · Esc exit · double-click a slider to reset it.
